== Installation ==

= Minimal Requirements =
* PHP: 5.x.x
* mySQL: 4.0, 4.1 or 5.x
* WordPress: 2.8 to 3.2.1

= Recommended Requirements =
* PHP: 5.2.x or newer
* mySQL: 5.x
* WordPress: 2.8 to 3.2.1

= Basic Installation =
1. Download and unzip the current version of the Apptha Banner plugin.
2. Transfer the entire appthabanner directory to your /wp-content/plugins/ directory
3. Activate the Apptha Banner plugin through the 'Plugins' menu in Admin Dashboard of WordPress
4. That's it! You're done.


== Other Notes ==

= To Edit the Apptha Banner Plugin =

1. Logon to Admin panel.
2. Settings -> Apptha Banner-> Banner Styles.
3. If you want to edit the styles or upload new image click one of the style and upload image by clicking the upload/edit tab.
4. You can sort the image order as your wish.
5. Bulk delete of image option is available, so you can delete the image if you don't want.

= How can i activate Apptha Banner plugin? =
   Download and unzip the current version of the Apptha Banner plugin,
   Transfer the entire appthabanner directory to your /wp-content/plugins/ directory,
   Activate the Apptha Banner plugin through the 'Plugins' menu in admin dashboard of WordPress.

= How can i add the banner in header? =
    Please Go To Admin Menu->Apperance->Editor and
    Select your theme in top select box and go to file header.php
    Add our <?php  if (function_exists('apptha_banner')) { apptha_banner(); } ?> before the  closing of header </div>

= How can i add the banner in post/pages? =
   When you don't like to have a banner on the header part of your blog, you can use the following plugin code to display banner on particular post/page.
   To insert Banner in Page/Post 
   Step 1: [appthabanner style=id]
   Step 2: In the above plugin code you have to provide banner style number as a value to 'id'. Example: [appthabanner style=1]
   Note: The style id is available under each banner style preview/thumb image on this page. (Next to Delete button)
